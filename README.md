# Arithmetic
## 四则运算测试器实现功能：
- √ 参与运算的操作数(operands)除了100以内的整数以外，还要支持真分数的四则运算。操作数必须随机生成。
- √ 运算符(operators)为 +, −, ×, ÷ 运算符的种类和顺序必须随机生成。
- √ 要求能处理用户的输入，并判断对错，打分统计正确率。
- √ 使用 -n 参数控制生成题目的个数。
### 附加功能:
- √ 支持带括号的多元复合运算
- √ 运算符个数随机生成（考虑小学生运算复杂度，范围在2~10）
